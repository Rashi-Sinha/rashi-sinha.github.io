Inverse Kinematics

Technical Work / IK System

Inverse Kinematics with Skinning

C++

Eigen

ADOL-C

OpenGL

This program takes in a 3D rigged character and implements Forward Kinematics, Tikhonov Regularization method for inverse kinematics, Linear Blend Skinning and Dual Quaternion Skinning. It is an interactive application where the user can pull on the IK handles and the joint angles and the character mesh moves accordingly.

Eigen library is used to solve large systems of linear equations and ADOL-C is used to solve for Jacobian matrix for IK.
