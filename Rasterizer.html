Rasterizer

This is a computer graphics program implementing a 3D rasterizer that reads in a user specified scene file and renders 2D images. Implemented linear expression evaluation (LEE), hidden surface removal with z-buffering, space transformations (from object to raster space), color interpolation, texture mapping with perspective correction and toon shading.

Coded in Python with scene files in JSON format. Note: The input scene files are not my own.

Some features are below:

Flat Shading

Gouraud Shading

Texture Mapping

Toon Shading

Phong Shading
