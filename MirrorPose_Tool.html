Tool Development

I've been working on a custom Python tool for Autodesk Maya that mirrors character poses across the YZ plane — designed to speed up animation workflows and support a variety of rig setups. It's still a work in progress, so I'm sharing what I've got so far . . .
While more advanced tools exist, I built this lightweight version to address a specific need I ran into while animating a walk cycle for a Technical Animation for Games class I am teaching. I wanted something accessible, simple and flexible — and that became the motivation behind this project.
Check out the demo and feature details below!

Tool Features

Current Features:

  1. Mirrors a pose across the YZ Plane.
    For middle/center controls, it simply inverts the transformations.
    For left/right controls (like legs), it swaps the transformations between left and right sides.
  2. Works on selected controls, allowing precise and non-destructive use.
  3. Compatible with a variety of rig setups via flexible naming convention support (e.g., L_, Left, _L).

Still in Progress:

  1. Working on refining mirroring logic for hand/arm controls.

Potential enhancements:

  1. Support mirroring across other planes.
  2. Add intuitive UI for mirror axis selections.
