A 3D third-person combat game in which the player is a blind martial artist with echolocation abilities.

My contribution as a Tech Artist in the team was to create a stylized manga-like environment shader for the scenes in the game that uses custom lighting from echolocation as a light source. The project involved implementing scriptable objects and writing shader code for custom lighting that was used for materials and visual effects.

This project is being developed by a team of students as part of the USC Games AGP course.
