Blindsight: War of The Wardens (Tech Art)

Unity 3D

C#

HLSL

URP Shader Graphs

VFX Graphs

Particle Systems

A 3D third-person combat game in which the player is a blind martial artist with echolocation abilities.

My contribution as a Technical Artist in the team was to create a stylized manga-inspired environment shader for the scenes in the game that uses custom lighting from echolocation as a light source. The project involved writing custom shader code, implementing scriptable objects for art-directable controls, designing shader graphs and creating combat visual effects.

Credits (MobyGames): This project was developed by a team of students as part of the USC Games AGP course.

Final Shader in Game
