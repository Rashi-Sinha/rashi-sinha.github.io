This is a standalone, cross-platform application I developed for technical game art students to streamline their media conversion workflows. Built with an intuitive PyQt interface, the tool replaces a previously Maya-dependent tool and allows artists to easily convert between image sequences and video files, as well as reformat videos for compatibility with Maya. (Github)

The application is packaged as a simple, double-click executable using PyInstaller and runs on both Windows and macOS, ensuring accessibility across all student workstations. It has been successfully used in student production pipelines to generate final animation videos from rendered image sequences and to convert reference footage into Maya-friendly formats for animation.

Tool Demo

Current Tool Features:

  1. Converts image sequences to video.
  2. Auto-plays converted videos.
  3. Allows adding multiple input sources and edit, reorder or delete them.
  4. Combines videos and image sequences into a single video.
  5. Converts video formats.
  6. Converts video to image sequences.

Potential enhancements:

  1. Add GIF conversions.
  2. Add "Help" section.
  3. Display output file Name in the Output Section.
  4. Clean-up output logs.
