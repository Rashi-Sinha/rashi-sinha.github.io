Jello Cube: A Mass-Spring Deformation System

C++

OpenGL

A physically-based simulation of a 3D elastic cube using a mass-spring system. The cube (unit size) is constrained inside a 2×2×2 bounding box with collision detection against the box and an optional inclined plane. Elastic behavior is controlled by spring constants from the input file, and an external force field can be applied throughout the volume that imapcts the simulation.

Supports Euler method or the RK4 time-step integration.

Input file specifies spring constants, integration method, inclined-plane parameters, force-field data, and initial vertex positions/velocities.
