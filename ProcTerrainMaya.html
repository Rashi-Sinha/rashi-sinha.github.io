Procedural Terrain Generation (Maya Tool)

Python

Maya

Perlin Noise

This is a tool created in Maya to add a plane with user inputs for subdivisions and modify its vertices to create a terrain using sampled from perlin noise. It also allows user to generate different terrains by choosing desired inputs to the noise function.

I developed this tool for a beginner level workshop that I held as part of the Women in Animation student chapter at USC. As a Lead, I wanted to bring exposure to the technical aspects of 3D content generation and potential career opportunities among interested students.
